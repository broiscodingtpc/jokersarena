Instrucțiuni pentru website-ul meu de gambling cu token Solana:

Tehnologii folosite:

HTML + CSS (sau TailwindCSS dacă vrei rapiditate)

JavaScript simplu (sau React pentru dinamism dacă se cere)

Comportament & Interacțiuni:

Toate butoanele sunt clickabile:

Telegram → deschide link: https://t.me/YOUR_GROUP

Whitepaper → deschide PDF: /whitepaper.pdf

Buy Token → duce către Raydium sau Jupiter presale

Butonul “ROLL” trebuie să simuleze un efect (glow, shake) și să aleagă un rezultat random (YOU WIN! / YOU LOSE!)

Afișează rezultatul într-un pop-up sau sub buton

(La MVP nu conectăm wallet, e doar frontend)

Design:

Font: ceva meme/graffiti style (ex: Bungee, Rubik Doodle Shadow)

Culori dominante: violet, neon orange, verde toxic

Responsive pe mobil

Folosește clase clare: .hero, .button, .flow, .popup, etc.

document.getElementById('roll-btn').addEventListener('click', () => {
  const resultDiv = document.getElementById('result');
  resultDiv.textContent = 'Rolling...';

  setTimeout(() => {
    const win = Math.random() > 0.5;
    resultDiv.textContent = win ? '🎉 YOU WIN!' : '💀 YOU LOSE!';
    resultDiv.style.color = win ? 'lime' : 'red';
  }, 1000);
});
